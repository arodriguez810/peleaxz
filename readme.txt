1) Entrar a tools y cambiar las rutas para
	a) database.bat (solo si usas mongodb)
	b) httpserver.bat
	
2) Entrar a \admin\config.json
	Cambiar las configuraciones deseadas
	Para omitir conexiones a base de datos borrar el nodo que desee mongo,mssql o mysql del JSON

3) Ejecutar las herramientas 
	a) httpserver.exe 
	b) database.exe(en caso de querer mongodb)
