Getting Started #back to top

Install NodeJS: https://nodejs.org/en/download/

Download Source Code from: https://github.com/arodriguez810/dragonframework.git

Install any of these database managers for local testing:
  • MY SQL Create example database and run 10-bootstrap\dbs\mysql.sql script
  • SQL Server Create example database and run 10-bootstrap\dbs\mssql.sql script
  • Oracle Xg Create example database and run 10-bootstrap\dbs\oracle.sql script
